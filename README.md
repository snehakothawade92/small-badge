# SmallBadge – Digital Badge Verification System

A Flask-based web application to issue and verify digital badges and certificates.

## Features
- Admin dashboard
- Organization dashboard
- Student dashboard
- Issue digital badges
- Public badge verification
- Login and registration system
- MongoDB integration

## Tech Stack
- Python
- Flask
- MongoDB
- HTML, CSS, JavaScript

## Run locally

```bash
pip install -r requirements.txt
python app.py
```

Open http://127.0.0.1:5000 in your browser.
